# Claude 提示词模板

## 代码审查模板

```
@claude 审查这段代码，检查：
1. 是否符合 MISRA 规范
2. 是否有内存泄漏风险
3. 是否有性能问题
4. 是否有安全漏洞
5. 是否有代码异味
```

## 代码生成模板

```
@claude 生成一个函数，功能：
1. 输入：[输入参数]
2. 输出：[输出结果]
3. 算法：[算法描述]
4. 要求：[性能要求]
```

**示例：**
```
@claude 生成一个函数，功能：
1. 输入：uint8_t *data, uint16_t size
2. 输出：校验和
3. 算法：CRC32
4. 要求：高效、无分支
```

## 架构设计模板

```
@claude 设计 [系统/模块] 的架构：
1. 功能需求：[功能列表]
2. 性能要求：[性能指标]
3. 约束条件：[技术约束]
4. 输出：架构文档 + 模块划分
```

## 问题分析模板

```
@claude 分析这个问题：
1. 问题描述：[问题详情]
2. 环境信息：[环境配置]
3. 已尝试方案：[已尝试的方法]
4. 期望结果：[期望的解决方案]
```

## 文档生成模板

```

2. 使用示例

```
